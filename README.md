# gox

*Additional algorithms and data structures for Go*

* [iterx](./iterx): Extensions for [iter.Seq](https://pkg.go.dev/iter#Seq)
* [slicex](./slicex): Additional functions for slices, extending [slices package](https://pkg.go.dev/slices)
* [ops](./ops): Function for intrinsic operators and predicates
* [types](./typex): Additional types constraints, like [golang.org/x/exp](https://pkg.go.dev/golang.org/x/exp/constraints) but stable
